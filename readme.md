# SensorFlow: 大语言模型增强的智能布点决策系统 (桌面端)

![Platform](https://img.shields.io/badge/Platform-Windows%2064--bit-blue.svg)
![Version](https://img.shields.io/badge/Release-v1.0.0-success.svg)
![Type](https://img.shields.io/badge/Type-Desktop%20Application-orange.svg)

> 本软件是一款专为市政工程师打造的**免安装桌面级决策支持系统**。系统融合了“大语言模型语义解析（LLM）”与“物理信息图神经网络（PI-GNN）”，能够根据工程师的自然语言需求，在毫无监测真值的全新管网中，一键秒级生成科学、安全、符合最优实验设计的传感器布点方案。

## 软件核心亮点

* **“听得懂人话”的交互设计**
  无需繁琐的界面参数配置。你可以直接在聊天框输入（如：“*预算有限，只需要选15个点，避开倒流区和深度大于5米的危险井*”），系统内置的 AI 中枢会自动将你的口语转化为底层的工程数学约束。
* **极速零样本推断寻优**
  彻底告别传统水力模型漫长的迭代计算。底层搭载的 PI-GNN 物理引擎，仅需读取静态 INP 拓扑文件，即可在秒级推断出全局水力模式的高维特征。
* **自适应科学排兵布阵**
  无需人工干预，自动算出最适合该管网的测点数量，并确保证监测点位在空间上科学合理。
* **一键生成可审计工程报告**
  自带创新的 TICI（拓扑信息捕获指数）评估体系。一键导出图文并茂的 PDF 决策报告，直接为工程投标或方案汇报提供量化的数据支撑。

## 下载与安装 (Installation)

本软件为免安装绿色版，下载后开箱即用：

1. 前往本仓库右侧的 **[Releases](#)** 页面。
2. 下载最新版本的 `SensorFlow_v1.0.0_Windows_x64.zip` 压缩包。
3. 解压到你的电脑中（建议解压到全英文路径下，如 `D:\SensorFlow`）。
4. 进入文件夹，双击运行 **`SensorFlow.exe`** 即可启动系统。

*(注：初次启动时，系统需要将底层 AI 神经网络载入显存/内存，可能需要等待 10-15 秒，请耐心等待。)*

## 快速上手指南 (Quick Start)

1. **导入模型**：点击软件左上角的 `载入 INP 模型`，选择你的管网数据，右侧画板会自动渲染管网高清拓扑图。
2. **框选区域（可选）**：如果有框选区域的需求，可以输入（如右上角部分），然后在地图上拖拽鼠标，圈定你想要重点排查的汇水子区域。
3. **输入意图**：在左侧输入框用自然语言描述你的工程约束（如管径限制、安装深度限制等）。
4. **智能寻优**：点击 `智能生成布点方案`，系统将自动进行合规性拦截与图寻优，并在地图上用高亮星号标记推荐点位。
5. **导出报告**：点击 `生成详细决策报告`，自动保存 PDF 文件到本地。

## 注意事项与环境要求

* **操作系统**：Windows 10 / 11 (64-bit)。
* **API 配置**：软件的自然语言解析功能依赖大模型 API 支持。请在软件设置界面/或配置文件中填入你个人的 API Key。
* **脱机运行**：底层的 PI-GNN 寻优引擎完全在本地离线运行，确保您的管网机密数据不会上传至任何云端。
